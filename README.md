
# Airborne Auto
A Cellular Automata for Airborne Diseases/一个空气传播传染病的元胞自动机模拟器    
元胞自动机模拟是对每个个体的仿真，不同于基于数据的曲线拟合。可以获得个体具体活动与疾病传播的关系。
